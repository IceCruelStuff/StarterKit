# StarterKit
A StarterKit plugin where you can insert your starter items in the Config that a new player can get
In the Config you can easily change the items yourself

# Config
Item0:
 -  id: 272 <- for the Items
 -  damage: 0 <- the damage Level
 -  count: 1 <- for the lot
